Folder Structure

General HTML structure is the same throughout the template. Here is the Folder structure.

HTML Folder structure for frest assets.

                
                    Theme/
                    ├── html/
                    |   ├── vertical-menu-template/
                    |   ├── vertical-menu-template-dark/
                    |   ├── vertical-menu-template-semi-dark/
                    |   ├── vertical-menu-boxicons-template/
                    |   ├── horizontal-menu-template/
                    |   └── horizontal-menu-template-dark/

Following is the details of what all html folders.

# Folder/Files Details
1 vertical-menu-template vertical-menu-template/ folder contains classic layout for frest admin.
2 vertical-menu-template-dark vertical-menu-template-dark/ folder contains dark layout for frest admin.
3 vertical-menu-template-semi-dark vertical-menu-template-semi-dark/ folder contains semi dark layout for frest admin.
4 vertical-menu-boxicons-template vertical-menu-boxicons-template/ folder contains main-menu-sidebar layout with Boxicons for frest admin.
5 horizontal-menu-template horizontal-menu-template/ folder contains horizontal layout for frest admin.
6 horizontal-menu-template-dark horizontal-menu-template-dark/ folder contains horizontal dark layout for frest admin.

Header Structure

General HTML structure is the same throughout the template. Here is the header structure.

                
                    <nav class="header-navbar main-header-navbar navbar-expand-lg navbar navbar-with-menu fixed-top">
                        <div class="navbar-wrapper">
                            <div class="navbar-container content">
                                <div id="navbar-mobile" class="navbar-collapse">
                                    .....
                                </div>
                            </div>
                        </div>
                    </nav>

Menu Structure

Generate frest menu using below menu structure.

                
                    <div class="main-menu menu-fixed menu-light menu-accordion menu-shadow expanded" data-scroll-to-active="true">
                        <!-- main menu header-->
                        <div class="navbar-header expanded">
                            .....
                        </div>
                        <!--/ main menu header-->
                        <!-- main menu content-->
                        <div class="main-menu-content">
                            <ul class="navigation navigation-main" id="main-menu-navigation" data-menu="menu-navigation" data-icon-style="">
                                .....
                            </ul>
                        </div>
                        <!--/ main menu content-->
                    </div>

Content Structure

Generate frest content area using following structure.

                
                    <div class="app-content content">
                      <div class="content-overlay"><div>
                        <div class="content-wrapper">
                            <div class="content-header row">
                                .....
                            </div>
                            <div class="content-body">
                                .....
                            </div>
                        </div>
                    </div>
